# Unity

[Unity](https://unity.com/) is a commercial game engine built on top of [Microsoft .Net](https://dotnet.microsoft.com/en-us/).
You can use [Nethereum](https://nethereum.com/), a .Net library for Ethereum, and MUD, support.

[There is also an older integration built by Emergence team](https://github.com/emergenceland/UniMUD).
