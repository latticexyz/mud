_Interface for the Module system.
A module can be installed within the context of a world, either as a root or non-root module.
This interface adheres to the ERC-165 standard for determining interface support._

## IModuleErrors

[Git Source](https://github.com/latticexyz/mud/blob/main/packages/world/src/IModuleErrors.sol)

This interface includes errors for the Module library.

_We bundle these errors in an interface (instead of at the file-level or in their corresponding library) so they can be inherited by IWorldKernel.
This ensures that all errors are included in the IWorldKernel ABI for proper decoding in the frontend._
