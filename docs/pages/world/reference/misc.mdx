### Functions

#### systemNamespace

Fetches the namespace of the current system.

_This function determines the system's namespace based on its interaction with the store. If the system is a root system, it returns the root namespace (an empty string). Otherwise, it retrieves the namespace using the system's registry.
This function must be used within the context of a system (either directly or within libraries called by a system)._

```solidity
function systemNamespace() internal view returns (bytes14);
```

**Returns**

| Name     | Type      | Description                                                 |
| -------- | --------- | ----------------------------------------------------------- |
| `<none>` | `bytes14` | Returns a bytes14 representation of the system's namespace. |

## revertWithBytes

[Git Source](https://github.com/latticexyz/mud/blob/main/packages/world/src/revertWithBytes.sol)

Utility function to revert transactions with raw bytes.

Reverts the transaction using the provided raw bytes as the revert reason.

_This can be especially useful when reverting with a message obtained from a low-level call or a pre-encoded error._

_Uses assembly to perform the revert operation with the raw bytes._

```solidity
function revertWithBytes(bytes memory reason) pure;
```

**Parameters**

| Name     | Type    | Description                  |
| -------- | ------- | ---------------------------- |
| `reason` | `bytes` | The raw bytes revert reason. |
