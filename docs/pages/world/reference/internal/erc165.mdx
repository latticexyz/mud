## ERC165Checker

[Git Source](https://github.com/latticexyz/mud/blob/main/packages/world/src/ERC165Checker.sol)

_Library used to query support of an interface declared via [IERC165](./erc165-external#ierc165).
Note that these functions return the actual result of the query: they do not
`revert` if an interface is not supported. It is up to the caller to decide
what to do in these cases._
